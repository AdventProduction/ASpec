# Предисловие
ASpec - Плагин, позволяющий следить за игроками.

**Версия: 1.16.5-1.21.4 (Paper)**

# Функционал
Выдав нужный пермишн хелперу, он сможет наблюдать за возможными нарушителями на сервере. 

# Конфиг
Базовый конфиг представлен ниже
```
messages:
  noPerm: "&cУ вас нет прав"
  noPlayer: "&cИгрок не онлайн"
  onlyPlayer: "&cКоманду может использовать только игрок"
  specStart: "&aВы начали слежку за игроком {target}"
  emptyPlayer: "&cВы не ввели ник!"
  noSpec: "&cВы не наблюдаете!"
  forYourself: "&cНельзя наблюдать за собой"
  banPlayer: "&cЗа этим игроком слежка запрещена!"
  reload: "&aПлагин перезагружен"
  usage:
    - "&eИспользование:"
    - "&e  /spec start ник"
    - "&e  /spec stop"

settings:
  distance: 25 #Расстояние, преодолев которое наблюдателя снова телепортирует к игроку
  blacklist:  #Никнеймы игроков, за которыми наблюдение запрещено
    - "Steve"
    - "Notch"
  blocked-commands:  #Команды, которые будут запрещены (если имеются аргументы, то указывать начало команды). Обязательно с "/"
    - "/tpa"
    - "/msg"
  atTheStart:  #Команды, которые будут отправляться в консоль, когда модератор начнет наблюдение
    - "vanish {player}"
  uponCompletion:  #Команды, которые будут отправляться в консоль, когда модератор завершит наблюдение
    - "vanish {player}"
    - "tell {player} хорошей игры"
```

# Возможности
После того как вы выдадите пермшн, указанный ниже, игрок сможет прописать команду для слежения, и будет перемещен в режим наблюдателя, а так же телепортирован к тому, за кем следит. После завершения слежки, наблюдающий будет перемещен на свое прежнее место.
Если вдруг плагин или сервер выключится, а хелпер будет в режиме наблюдения, то он так же будет перемещен на свое прежнее место. В режиме наблюдателя хелпер не сможет телепортироваться к другим игрокам через ивентарь, а так же не сможет отлететь далеко от цели, дистанция задается в конфиге.
Еще в конфиге можно настроить сообщения, список игроков, за которыми нельзя следить, запрещенные команды, исполняющиеся команды во время слежки и отмены слежки (например если нужно выдать ваниш). В сообщениях возможно использование HEX цветов. 

# Кратко о командах и разрешениях

> ### Команды
**/spec reload** - Перезагрузить плагин.\
**/spec start ник** - Начать слежение.\
**/spec stop** - Остановить слежение.

> ### Разрешения
**aspec.*** - Доступ ко всем командам.\
**aspec.use** - Доступ к командам /spec start/stop\
**aspec.reload** - Доступ к /spec reload
